```
Design an application settings screen.

Layout:

1. Header: Back arrow, "Settings"

2. Settings sections (vertical list of grouped items):

Section A: "Tagging Defaults"
- Default tagging mode: Toggle "Essential" / "Full"

  - Full: "5+ questions with diagnostics"

- Default tagging speed dropdown: "0.25x"
- Default fast forward speed dropdown: "1x"

- Toggle: "Serve diagnostics" (on/off)
- Toggle: "Receive diagnostics" (on/off)
- Toggle: "Third ball diagnostics" (on/off)
- Toggle: "Unforced error diagnostics" (on/off)
- Note: "These options only apply in Full tagging mode"

Section C: "Account"
- Display: Email address
- "Sign Out" button

Section D: "Sync"
- Status indicator: "Last synced: 2 minutes ago"
- "Sync Now" button

Section E: "About"
- App version: "1.0.0"
- "Privacy Policy" link
- "Terms of Service" link

Color scheme: Dark theme, section headers in muted text, toggles with teal accent when active

Typography: Section headers in caps small text, item labels clear

Style: Standard settings pattern, grouped sections, easy to scan and toggle
```
