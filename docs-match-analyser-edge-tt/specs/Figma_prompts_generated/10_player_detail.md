# 10 - Player Detail

## Screen Purpose
View/edit a single player

---

## Figma AI Prompt

```
Design a player detail/edit screen.

Layout:

1. Header: Back arrow, "Edit Player" (or "New Player" for creation mode)

2. Form content (centered card, 480px max-width):

A. Basic Info:
- Name input (text field)
- Rating slider (1-10) with current value displayed

B. Recent Matches section:
- List of 3-5 most recent matches involving this player
- Each shows opponent, score, date, tagging mode badge
- "View All Matches" link

C. Quick Stats (read-only summary):
- Total matches played
- Win rate %
- Serve fault rate %
- Most common shot type

3. Actions (bottom):
- "Save Changes" primary button
- "Delete Player" destructive button (with confirmation)

Color scheme: Dark theme, form inputs with clear borders, destructive action in red

Typography: Labels above inputs, rating number large

Style: Simple edit form, essential information only
```
